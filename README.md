# To-Do List

O To-Do List é uma aplicação web desenvolvida com HTML, CSS e JavaScript puro, que permite ao usuário gerenciar tarefas de forma simples e interativa. A aplicação possibilita adicionar, remover e organizar itens dinamicamente, atualizando a interface em tempo real conforme as interações do usuário.

## Funcionalidades

- Adição de novas tarefas

- Remoção de tarefas da lista

- Atualização dinâmica da interface

- Manipulação direta do DOM

- Estrutura organizada sem uso de frameworks

## Objetivo do Projeto

O projeto foi desenvolvido com o objetivo de consolidar fundamentos essenciais do desenvolvimento frontend, incluindo:

- Manipulação de eventos

- Controle de estado da aplicação

- Estruturação lógica de funcionalidades

- Separação básica entre estrutura (HTML), estilo (CSS) e comportamento (JavaScript)

# Tecnologias Utilizadas

- HTML5

- CSS3

- JavaScript
